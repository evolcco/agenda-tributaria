Dataset público

Agenda Tributária

Dados oficiais da Receita Federal do Brasil, normalizados em JSON e ICS. Atualizados automaticamente a cada mês a partir da planilha publicada na agenda tributária federal.

Endpoints

  • data/index.json — índice de meses disponíveis
  • data/agenda-AAAA-MM.json — payload por mês (schema 1.1.0)
  • data/agenda-AAAA-MM.ics — calendário no padrão RFC 5545

Exemplo de consumo

const index = await fetch('https://<owner>.github.io/agenda-tributaria/data/index.json')
  .then(r => r.json());

const ultimo = index.months[0];
const agenda = await fetch(`https://<owner>.github.io/agenda-tributaria/data/${ultimo.file}`)
  .then(r => r.json());

console.log(`Agenda de ${ultimo.label}: ${agenda.counts.tributos} tributos`);

O que tem cada item

  • due_date — data publicada pela RFB (ISO)
  • effective_due_date — data ajustada para dia útil (Lei 9.430/96 art. 18)
  • adjusted e adjustment_reason — quando houve prorrogação e por quê

Atualização

Um GitHub Action roda nos dias 1 e 15 de cada mês, baixa a planilha mais recente, regenera os JSONs e ICS e abre um PR. Após o merge, esta página é republicada automaticamente.
